# Revenue vs Paid Search Cost Analysis

SQL analysis of monthly revenue and paid search costs to evaluate marketing performance and compare income vs advertising spend over time.

## Project Overview

This project analyses monthly revenue and paid search costs to evaluate marketing performance over time.

## Business Problem

Marketing teams need to understand whether their advertising spend generates enough revenue.
This analysis helps compare income and costs every month.

## Data Sources

* `product` — product prices
* `order` — customer purchases
* `session` — session dates
* `paid_search_cost` — advertising costs

## Metrics

* Revenue
* Paid search cost

## What this analysis shows

* Comparison of revenue and marketing spend over time
* Identification of trends in performance
* Basis for evaluating marketing efficiency

## SQL File

The analysis is implemented in:

`revenue_vs_paid_search_cost.sql`

## Skills Demonstrated

* SQL joins
* Aggregations
* UNION ALL
* Working with dates
